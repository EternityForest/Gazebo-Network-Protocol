===== Contents =====

==== GazeboTemplate ====
Not an actual openoffice template but a blank text file with all the styles and fonts set up. Requries Linux Libertine and Quivira, two completely free software fonts. It is very easy to make yourself a template from the blank file if you so choose.
Created for use in documentation.

==== ElectronicsSpreadsheet ====
A spreadshhet originally designed for physical layer calculations with many different calculators availible.
It expanded a bit to be a more  general electronics tool.
